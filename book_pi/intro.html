Permanent income model¶

This notebook serves as an example from [Hansen and Sargent, 2021]. It displays computational results of a permanent income model. This model is a version of Friedman’s permanent income model, where habit persistence preferences, motivated by Hansen, Sargent, and Tallarini Jr [1999], is considered. The model is approximately solved using a small noise expansion technique as in Lombardo and Uhlig [2018].

This notebook is mainly for illustration of model specific results. If interested in more general computation details, users can refer to the Computational Details notebook accompanying this notebook.
